Teknik Sağlık Notları (25 Mayıs Deploy Sonrası)

Sağlık metrikleri belirgin iyileşti. 25 Mayıs 13:28 UTC'de uygulanan teknik fix paketi sonrasında son 1 saatlik canlı log durumu: 4xx oranı %0.4 (önceki %2.2'den ↓), 5xx oranı %0.3 (önceki %2.2'den ↓).
  • Search Console'da görülen 18 sayfa 5xx (/sitemap_index.xml, /sitemap-0.xml) için 301 redirect eklendi.
  • Search Console'da görülen 27 sayfa 404 (eski /urunler/kategori/*, /kategori/*, /blog/kategori/* URL'leri) için 301 redirect eklendi.
  • Soft 404 olan auth/sepet kalıntı sayfaları (/login, /profile, /sepet, /checkout, /wp-admin) yeni hedeflere yönlendirildi.
  • Admin panel /api/auth/token 404 sorunu giderildi: resolveBaseUrl() prod fallback /api → /api/v1 ve rewrite stripping mantığı düzeltildi.
  • Admin panel logo yolu (/uploads/media/logo/vistaseed_logo.png) 404'leri çözüldü.
  • Backend için VPS MySQL app@localhost hesabına eksik UPDATE / INSERT yetkileri verildi (refresh_token akışı sorunsuz çalıştı).

Henüz çözülmemiş tespitler

  • Eski tekil URL pattern: /tr/urun/<slug> (50+ adet 404) — redirect kuralı önerildi, henüz eklenmedi.
  • Sitede mevcut olmayan /tr/grup-sirketlerimiz/* URL'leri (20+ adet 404) — dış kaynaktan yanlış backlink, redirect veya 410 önerildi.
  • apple-touch-icon-precomposed.png eksik (12 adet 500); admin panel robots.txt eksik (29 adet 404).
